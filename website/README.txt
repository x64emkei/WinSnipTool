==============================
WinSnipTool Website
==============================

This folder contains the complete WinSnipTool website.
Single-page design with Tailwind CSS (CDN).

MAIN FILE:
----------
index.html              Complete website (all CSS/JS inline)

IMAGES:
-------
banner_main.png         Main header banner (980×180)
icon.png                WinSnipTool icon (96×96)

Gallery Images:
image1_2000.png         Windows 2000 screenshot
image2_xp.png           Windows XP screenshot
image3_2003.png         Windows Server 2003 (MISSING - see MISSING_IMAGES.txt)

Control Panel Images:
cp1.png                 Main toolbar interface
cp2.png                 Settings panel (MISSING - see MISSING_IMAGES.txt)
cp3.png                 Advanced options (MISSING - see MISSING_IMAGES.txt)

Legacy Images (can be removed if not needed):
2k_about.png            Original Windows 2000 About dialog
xp_about.png            Original Windows XP About dialog
toolstrip.png           Original toolbar screenshot

DEPLOYMENT:
-----------
Upload all files to web server:
- index.html
- All .png image files

The website loads Tailwind CSS from CDN (no local CSS files needed).

CUSTOMIZATION:
--------------
Edit index.html to customize:
- Colors (tailwind.config section)
- Content (HTML sections)
- Images (replace PNG files)

TESTING:
--------
Open index.html in any modern browser.
Works offline (except Tailwind CDN requires internet).

COMPATIBILITY:
--------------
- Modern browsers (Chrome, Firefox, Edge, Safari)
- Mobile responsive
- Keyboard accessible
- Screen reader friendly
